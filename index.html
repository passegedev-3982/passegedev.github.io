Frequently Asked Questions

The game won't launch. What should I do?

Make sure your drivers are up to date and that your system meets the minimum requirements. Try verifying the game files via Steam (Right-click the game → Properties → Local Files → Verify integrity of game files).

My save file is missing or corrupted.

Save files are stored in %AppData%\[GameName] on Windows. If a save is corrupted, you can delete the file there and start fresh. We recommend backing up this folder regularly.

The game crashes mid-run. How do I report it?

Please send a bug report to the contact email below with your system specs and a description of what happened. If possible, include the output_log.txt from the game's installation folder.

I found a bug or want to give feedback.

All bug reports and feedback are welcome! Use the contact form below or reach out via the Steam community hub. Your input directly shapes development.
